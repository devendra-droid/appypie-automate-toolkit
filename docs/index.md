# Appy Pie Automate Toolkit

Welcome to the official documentation for the **Appy Pie Automate Toolkit** — an open-source SDK and toolset for [Appy Pie Automate](https://appypieautomate.ai), the no-code workflow automation platform connecting 1,000+ apps.

## What is Appy Pie Automate?

[Appy Pie Automate](https://appypieautomate.ai) is a powerful no-code automation platform that lets you connect your favorite apps and automate repetitive workflows — without writing a single line of code.

**Key features:**

- ✅ **1,000+ app integrations** — Gmail, Slack, Shopify, HubSpot, Salesforce, Stripe, GitHub, Jira, Notion, Airtable, and more
- ✅ **No-code workflow builder** — drag-and-drop interface, no programming required
- ✅ **Multi-step automations** — chain multiple actions in a single workflow
- ✅ **Real-time triggers** — instant webhooks and polling triggers
- ✅ **Free tier available** — start automating for free
- ✅ **Trusted by 10M+ users worldwide**

## Quick Start

👉 **[Sign up for free at appypieautomate.ai](https://accounts.appypie.com/register)**

### Install the JavaScript SDK

```bash
npm install appypie-automate
```

### Install the Python SDK

```bash
pip install appypie-automate-toolkit
```
